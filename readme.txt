This plugin for WooCommerce products, adds options under your Linked Products options, which would allow you to choose custom related products. If you keep it empty, a random product from the same category would show.

== Description ==
This plugin for WooCommerce products, adds options under your Linked Products options, which would allow you to choose custom related products. If you keep it empty, a random product from the same category would show.

== Installation ==


    Upload Simple Related Products for WooCommerce to the /wp-content/plugins/ directory

    Activate the plugin through the 'ËœPlugins' menu in WordPress
